## **Реализованные функции**

1.**double* full_elements(double* ptr_array, int n)**

Назначение: Заполнение массива значениями функции sin(2x)+cos(x)

Параметры: 

ptr_array - указатель на массив
n - размер массива
Возвращает: указатель на заполненный массив




2. int put_elements(double *ptr_array, int n)

Назначение: Печать элементов массива в форматированном виде

Параметры:

ptr_array - указатель на массив

n - размер массива

Возвращает: количество выведенных элементов







3. double* calc_elements(double *ptr_array, int n)

Назначение: Обработка элементов массива (возведение каждого элемента в квадрат)

Параметры:


ptr_array - указатель на массив

n - размер массива

Возвращает: указатель на обработанный массив





4. double sum_elements(double *ptr_array, int begin, int end)

Назначение: Вычисление суммы элементов массива в заданном диапазоне

Параметры:


ptr_array - указатель на массив

begin - начальный индекс (включительно)

end - конечный индекс (включительно)

Возвращает: сумму элементов

5. int find_element(double *ptr_array, int n, double element)

Назначение: Поиск элемента в массиве

Параметры:


ptr_array - указатель на массив

n - размер массива

element - искомое значение

Возвращает: индекс найденного элемента или -1 если не найден

6. double find_max_before_k(double *ptr_array, int n, int K)

Назначение: Нахождение максимального элемента до позиции K

Параметры:


ptr_array - указатель на массив

n - размер массива

K - позиция, до которой искать (включительно)

Возвращает: максимальное значение или -1 при ошибке

7. int count_positive_elements(double *ptr_array, int n)

Назначение: Подсчет положительных элементов в массиве

Параметры:


ptr_array - указатель на массив

n - размер массива

Возвращает: количество положительных элементов

8. double calculate_average(double *ptr_array, int n)

Назначение: Вычисление среднего арифметического элементов массива

Параметры:


ptr_array - указатель на массив

n - размер массива

Возвращает: среднее значение

9. double find_min_element(double *ptr_array, int n)

Назначение: Нахождение минимального элемента в массиве

Параметры:


ptr_array - указатель на массив

n - размер массива

Возвращает: минимальное значение

10. double* sort_array(double *ptr_array, int n)

Назначение: Сортировка массива по возрастанию (пузырьковая сортировка)

Параметры:


ptr_array - указатель на массив

n - размер массива

Возвращает: указатель на отсортированный массив

11. double* reverse_array(double *ptr_array, int n)

Назначение: Реверс массива (переворот порядка элементов)

Параметры:


ptr_array - указатель на массив

n - размер массива

Возвращает: указатель на перевернутый массив

12. double* copy_array(double *src, double *dest, int n)

Назначение: Копирование массива

Параметры:


src - исходный массив

dest - целевой массив

n - размер массивов

Возвращает: указатель на скопированный массив

13. double func(double x)

Назначение: Вычисление значения функции по варианту

Параметры:


x - аргумент функции

Возвращает: значение функции f(x) = e^(-0.1x) * sin(x)
